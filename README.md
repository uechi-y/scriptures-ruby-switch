# 聖典ふりがな表示切り替え

これは、[末日聖徒イエス・キリスト教会](https://www.churchofjesuschrist.org/?lang=jpn) の公式サイトで聖典を学習する際に、ふりがな（ルビ）の表示・非表示を切り替えるための Google Chrome 拡張機能です。

ふりがなが不要なユーザーにとって、より快適な読書環境を提供します。

## 主な機能

*   **ふりがなの表示切り替え:**
    聖典ページのサイドパネル（表示オプション）に、「ふりがなを表示する」トグルスイッチを追加します。

*   **設定の保存:**
    スイッチのオン・オフ設定は自動的に保存され、次回以降も同じ設定が維持されます。

*   **日本語ページ限定:**
    URLに `lang=jpn` が含まれる日本語のページでのみ動作します。他の言語のページには影響を与えません。

## 使い方

1.  [末日聖徒イエス・キリスト教会](https://www.churchofjesuschrist.org/?lang=jpn)のサイトで、言語が日本語に設定された聖典ページ（例: [モルモン書](https://www.churchofjesuschrist.org/study/scriptures/bofm?lang=jpn)）を開きます。
2.  画面の右側にある「選択肢」のアイコン（点が縦に3つ並んだマーク）をクリックし、サイドパネルを開きます。
3.  パネル内の「設定」から「表示オプション」を選択します。
4.  「ふりがなを表示する」スイッチを操作することで、リアルタイムにふりがなの表示・非表示を切り替えることができます。
